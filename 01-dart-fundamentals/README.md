# 01 - Dart Fundamentals

Pure Dart - no Flutter yet. Get comfortable with the language itself first.

## Concepts covered
- Variables, null safety (`?`, `!`, `late`)
- Functions (named params, optional params, arrow functions)
- Classes, constructors, inheritance, mixins
- Collections (List, Map, Set) and common methods (map/where/fold)
- Futures and async/await (Dart's async model, same concepts as JS Promises)

## Files
- `variables_null_safety.dart`
- `functions.dart`
- `classes_oop.dart`
- `collections.dart`
- `async_futures.dart`

## Run it
```bash
dart run variables_null_safety.dart
dart run functions.dart
dart run classes_oop.dart
dart run collections.dart
dart run async_futures.dart
```
